# flutter camera

นี่คืองานตัวอย่างที่สร้างด้วย Flutter

## ฟีเจอร์หลัก
- ถ่ายภาพ
- สลับกล้องหน้าและหลัง
- บันทึกภาพลงแกลเลอรี

## วิธีการติดตั้ง

### 1. โคลนโครงการนี้

```bash
git clone https://github.com/thanachok11/flutter_camera.git
```

### 2. ติดตั้ง dependencies
 ```bash
flutter pub get
```
